# agenticraft_foundation.verification.probabilistic

Probabilistic verification for Discrete-Time Markov Chains (DTMC).

Provides DTMC representation, reachability probability computation,
steady-state distribution, and expected steps analysis. Based on
Hansson & Jonsson (1994) and Baier & Katoen Ch. 10.
