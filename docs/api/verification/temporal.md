# agenticraft_foundation.verification.temporal

CTL (Computation Tree Logic) temporal model checking over LTS.

Provides a formula AST (Atomic, Not, And, Or, EX, EF, EG, EU, AX, AF, AG, AU)
and bottom-up model checking via backward fixpoint computation. Based on
Clarke & Emerson (1981) and Baier & Katoen Ch. 4 & 6.
